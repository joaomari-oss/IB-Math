Módulo 1

Funções (Functions)

O conceito mais fundamental: uma regra que associa cada entrada a exatamente uma saída

Conceito

O que é uma Função?

Uma função (function) é uma relação matemática que associa cada elemento de um conjunto (chamado domínio) a exatamente um elemento de outro conjunto (chamado contradomínio).

💡
Pensando como professor: Imagine uma máquina — você coloca um número (entrada), a máquina aplica uma regra, e devolve exatamente um resultado (saída). Essa "máquina" é uma função!
Notação
f(x) = expressão

Lê-se: "f de x". O x é a variável independente (entrada) e f(x) é a variável dependente (saída).

Domínio (Domain)

Conjunto de todos os valores possíveis de entrada (x)

Contradomínio (Range)

Conjunto de todos os valores possíveis de saída (f(x))

Imagem (Image)

O valor de saída para uma entrada específica

Exemplo

Dada a função f(x) = 2x + 3

  • Se x = 1 → f(1) = 2(1) + 3 = 5
  • Se x = 4 → f(4) = 2(4) + 3 = 11
  • Domínio: todos os números reais (ℝ)
  • Imagem: todos os números reais (ℝ)
Interativo

Teste uma Função

Digite uma expressão e um valor de x para calcular f(x)

Resultado: f(5) = 13
Conceito Importante

Teste da Reta Vertical (Vertical Line Test)

Para verificar se um gráfico representa uma função, aplique o teste da reta vertical: se qualquer reta vertical traçada no gráfico tocar a curva em no máximo um ponto, então o gráfico é uma função.

💡
Dica: Um círculo não é uma função, pois uma reta vertical pode cruzá-lo em dois pontos. Já uma parábola é uma função!
Módulo 2

Funções Lineares (Linear Functions)

A forma mais simples de função: uma relação constante entre variáveis

Conceito

A Equação da Reta

Uma função linear (linear function) gera um gráfico em linha reta. A taxa de variação é constante — ou seja, para cada aumento de 1 em x, f(x) aumenta sempre a mesma quantidade.

Forma Geral
f(x) = mx + c

Onde m = inclinação (slope/gradient) e c = intercepto-y (y-intercept)

Inclinação (Gradient)

m = (y₂ − y₁) / (x₂ − x₁)

Mede o quão inclinada é a reta. Se m > 0 a reta sobe; se m < 0 a reta desce; se m = 0 a reta é horizontal.

Intercepto-y (y-intercept)

c = f(0)

O ponto onde a reta cruza o eixo y. É o valor de f(x) quando x = 0.

💡
Pensando como professor: A inclinação é como a velocidade — diz o quão rápido y muda quando x muda. Um carro a 60 km/h tem "inclinação" 60 no gráfico distância × tempo!
Exemplo no Mundo Real

Uma loja cobra R$15,00 de taxa fixa + R$3,00 por km em entregas.

C(x) = 3x + 15

  • m = 3 → Custo aumenta R$3 por km
  • c = 15 → Taxa fixa de R$15
  • C(10) = 3(10) + 15 = R$45,00
Gráfico Interativo

Explore a Função Linear

Ajuste m e c para ver como a reta muda

f(x) = 1x + 0
Propriedades

Retas Paralelas e Perpendiculares

Paralelas (Parallel Lines)

m₁ = m₂

Duas retas são paralelas quando têm a mesma inclinação.

Perpendiculares (Perpendicular)

m₁ × m₂ = −1

Duas retas são perpendiculares quando o produto de suas inclinações é −1.

Módulo 3

Funções Quadráticas (Quadratic Functions)

Parábolas — curvas que modelam trajetórias, lucros, áreas e muito mais

Conceito

A Parábola

Uma função quadrática (quadratic function) é um polinômio de grau 2 cujo gráfico é uma parábola. Ela pode abrir para cima (∪) quando a > 0, ou para baixo (∩) quando a < 0.

Forma Geral
f(x) = ax² + bx + c

Onde a ≠ 0 determina a abertura; b influencia o deslocamento; c é o y-intercept

Forma do Vértice
f(x) = a(x − h)² + k

Onde (h, k) é o vértice da parábola

Vértice (Vertex)

Ponto mais alto (se a < 0) ou mais baixo (se a > 0). x = −b/(2a)

Eixo de Simetria (Axis of Symmetry)

Reta vertical passando pelo vértice: x = −b/(2a)

Discriminante (Discriminant)

Δ = b² − 4ac → determina o número de raízes

💡
Pensando como professor: O discriminante é como um "detector de raízes":
• Δ > 0 → duas raízes reais (a parábola cruza o eixo x duas vezes)
• Δ = 0 → uma raiz (a parábola toca o eixo x em um ponto)
• Δ < 0 → sem raízes reais (a parábola não toca o eixo x)
Fórmula de Bhaskara
x = (−b ± √Δ) / 2a

Onde Δ = b² − 4ac. Encontra as raízes (zeros) da função.

Exemplo no Mundo Real

Uma bola é lançada para cima com altura dada por:

h(t) = −5t² + 20t + 1

  • a = −5 → parábola abre para baixo (a bola sobe e depois desce)
  • Vértice: t = −20/(2×(−5)) = 2 → altura máxima em t = 2s
  • h(2) = −5(4) + 20(2) + 1 = 21 metros
  • h(0) = 1 metro (altura inicial)
Gráfico Interativo

Explore a Parábola

Ajuste a, b e c para ver como a parábola se transforma

f(x) = 1x² + 0x + 0
Vértice: (0, 0) | Δ = 0
Módulo 4

Funções Cúbicas (Cubic Functions)

Polinômios de grau 3 — curvas com forma de "S" e até dois pontos de inflexão

Conceito

Polinômios de Grau 3

Uma função cúbica (cubic function) é um polinômio de grau 3. Seu gráfico tem uma forma característica de "S" (ou "S" invertido), podendo ter até dois pontos de retorno (turning points) e até três raízes reais.

Forma Geral
f(x) = ax³ + bx² + cx + d

Onde a ≠ 0. Se a > 0 a curva sobe da esquerda para a direita; se a < 0 ela desce.

💡
Pensando como professor: A diferença fundamental entre quadrática e cúbica é o comportamento nos extremos:
• Quadrática: ambos os "braços" vão na mesma direção (↑↑ ou ↓↓)
• Cúbica: os "braços" vão em direções opostas (↙↗ ou ↗↙)
Pontos de Retorno (Turning Points)

Uma cúbica pode ter 0 ou 2 turning points (máximo local e mínimo local)

Ponto de Inflexão (Point of Inflection)

Onde a concavidade muda (de côncava para convexa ou vice-versa)

Raízes (Roots/Zeros)

Uma cúbica sempre tem pelo menos 1 raiz real, podendo ter até 3

Exemplo

Considere f(x) = x³ − 3x

  • Raízes: x = 0, x = √3, x = −√3
  • Máximo local em x = −1: f(−1) = 2
  • Mínimo local em x = 1: f(1) = −2
  • Ponto de inflexão em x = 0
Gráfico Interativo

Explore a Função Cúbica

Ajuste o coeficiente a para ver como a curva se comporta

f(x) = 1x³ + 0x² − 3x
Módulo 5

Funções Exponenciais (Exponential Functions)

Crescimento explosivo e decaimento — modelando populações, finanças e radioatividade

Conceito

Crescimento e Decaimento Exponencial

Uma função exponencial (exponential function) é aquela onde a variável aparece no expoente. Diferente das funções polinomiais, o crescimento (ou decaimento) é cada vez mais rápido.

Forma Geral
f(x) = a · bˣ

Onde a é o valor inicial (quando x = 0) e b é a base (fator de crescimento)

Crescimento (Growth)

b > 1

A função cresce exponencialmente. Ex: populações, juros compostos, vírus.

Decaimento (Decay)

0 < b < 1

A função decresce exponencialmente. Ex: radioatividade, depreciação, resfriamento.

💡
Pensando como professor: A diferença entre crescimento linear e exponencial é dramática. Se você ganha R$100/dia (linear), em 30 dias tem R$3.000. Mas se dobra a cada dia começando com R$0,01 (exponencial), em 30 dias tem mais de R$10 milhões!
Assíntota Horizontal (Horizontal Asymptote)

A curva se aproxima cada vez mais de y = 0 (ou outro valor), mas nunca toca

Meia-vida (Half-life)

Tempo necessário para a quantidade cair pela metade (usado em decaimento)

Exemplo no Mundo Real

Uma população de bactérias dobra a cada hora, começando com 100:

P(t) = 100 · 2ᵗ

  • P(0) = 100 (população inicial)
  • P(3) = 100 · 8 = 800 bactérias (após 3 horas)
  • P(10) = 100 · 1024 = 102.400 bactérias (após 10 horas)

f(x) = 1 · 2ˣ
Tipo: Crescimento (b > 1)
Módulo 6

Funções Inversas (Inverse Functions)

Desfazendo a função — voltando da saída para a entrada

Conceito

O que é uma Função Inversa?

A função inversa (inverse function) de f, denotada f⁻¹, é a função que "desfaz" o que f faz. Se f transforma a em b, então f⁻¹ transforma b de volta em a.

Propriedade Fundamental
f(f⁻¹(x)) = x   e   f⁻¹(f(x)) = x

Aplicar f e depois f⁻¹ (ou vice-versa) retorna ao valor original

💡
Pensando como professor: Se f(x) = 2x (dobrar), então f⁻¹(x) = x/2 (dividir por 2). A inversa é como o botão "desfazer" no computador!

Como encontrar f⁻¹(x):

1
Escreva y = f(x)
2
Troque x por y e y por x
3
Isole y (resolva para y)
4
Renomeie y como f⁻¹(x)
Exemplo Passo a Passo

Encontre a inversa de f(x) = 3x + 6

  1. y = 3x + 6
  2. x = 3y + 6 (trocamos x e y)
  3. x − 6 = 3y → y = (x − 6)/3
  4. f⁻¹(x) = (x − 6)/3

Verificação: f(2) = 3(2) + 6 = 12 → f⁻¹(12) = (12−6)/3 = 2 ✓

Gráfico

Reflexão sobre y = x

O gráfico de f⁻¹ é a reflexão do gráfico de f em relação à reta y = x. Isso significa que se o ponto (a, b) está no gráfico de f, então o ponto (b, a) estará no gráfico de f⁻¹.

Gráfico Interativo

Visualize a Inversa

Veja f(x) e f⁻¹(x) refletidas sobre y = x

f(x) = 2x + 1  |  f⁻¹(x) = (x − 1)/2
f(x) f⁻¹(x) y = x
Módulo 7

Transformações de Funções (Transformations)

Transladar, refletir, esticar e comprimir gráficos sistematicamente

Conceito

Transformações Geométricas de Gráficos

Dado um gráfico de y = f(x), podemos criar novos gráficos aplicando transformações. No IB, as quatro transformações fundamentais são: translações (shifts), reflexões, estiramentos (stretches) e compressões.

💡
Regra de Ouro do IB: Transformações "dentro" da função (que afetam x) são contra-intuitivas — agem na direção oposta. Transformações "fora" da função (que afetam y) funcionam como esperado.

Translação vertical: y = f(x) + d

d > 0 → move para cima; d < 0 → move para baixo.

Translação horizontal: y = f(x − c)

c > 0 → move para a direita; c < 0 → move para a esquerda. (Contra-intuitivo!)

No IB: A translação é representada pelo vetor (c, d) → desloca o gráfico c unidades horizontalmente e d unidades verticalmente.

Reflexão no eixo x: y = −f(x) → inverte todos os valores de y.

Reflexão no eixo y: y = f(−x) → espelha o gráfico horizontalmente.

Exemplo: Se f(x) = x², então −f(x) = −x² (parábola invertida) e f(−x) = (−x)² = x² (inalterado, pois a parábola é simétrica).

Estiramento vertical (fator a): y = a · f(x)

|a| > 1 → estica verticalmente; 0 < |a| < 1 → comprime verticalmente.

Estiramento horizontal (fator 1/b): y = f(bx)

|b| > 1 → comprime horizontalmente; 0 < |b| < 1 → estica horizontalmente. (Contra-intuitivo!)

Exemplo Passo a Passo — IB Style

Dado f(x) = x², descreva o gráfico de g(x) = 2(x − 3)² + 1:

  1. Translação horizontal: 3 unidades para a direita (x − 3)
  2. Estiramento vertical: fator 2 (a parábola fica mais estreita)
  3. Translação vertical: 1 unidade para cima (+1)

O vértice move de (0, 0) para (3, 1).

Resumo Geral
y = a · f(b(x − c)) + d

a = estiramento vertical + reflexão eixo x | b = estiramento horizontal + reflexão eixo y | c = translação horizontal | d = translação vertical

Módulo 8

Funções Compostas (Composite Functions)

Combinando funções — a saída de uma se torna a entrada de outra

Conceito

O que é Composição de Funções?

A composição (composition) de duas funções f e g, denotada (f ∘ g)(x) ou f(g(x)), significa aplicar g primeiro e depois aplicar f ao resultado.

Definição
(f ∘ g)(x) = f(g(x))

Lê-se "f composta com g de x". Aplique g primeiro, depois f.

💡
Pensando como professor: Imagine uma fábrica com duas máquinas em série. A Máquina G processa a matéria-prima e a Máquina F processa o que G produziu. O resultado final é f(g(x)).
Ordem importa!

Em geral, f(g(x)) ≠ g(f(x)). A composição não é comutativa.

Domínio da composição

x deve estar no domínio de g, e g(x) deve estar no domínio de f.

Exemplo Passo a Passo

Dadas f(x) = 2x + 1 e g(x) = x²

  • f(g(x)) = f(x²) = 2(x²) + 1 = 2x² + 1
  • g(f(x)) = g(2x + 1) = (2x + 1)² = 4x² + 4x + 1
  • Verificação: f(g(3)) = f(9) = 19 ✓ | g(f(3)) = g(7) = 49 ✓

Note que f(g(x)) ≠ g(f(x)), confirmando que a ordem importa!

Conexão: Composição + Inversa

Se f e f⁻¹ são inversas, então:

  • f(f⁻¹(x)) = x (a composição de uma função com sua inversa é a identidade)
  • f⁻¹(f(x)) = x

Isso é exatamente a definição de função inversa no currículo IB!

Módulo 9

Funções por Partes (Piecewise Functions)

Funções definidas por regras diferentes em intervalos diferentes

Conceito

O que é uma Função por Partes?

Uma função por partes (piecewise function) usa regras diferentes para diferentes intervalos do domínio. O gráfico pode ter "quebras", "saltos" ou mudar de formato em certos pontos.

Notação
        ⎧ regra₁,  se x < a
f(x) = ⎨ regra₂,  se a ≤ x < b
        ⎩ regra₃,  se x ≥ b

Cada "parte" tem seu próprio domínio e expressão.

💡
No IB: Questões sobre piecewise functions frequentemente pedem para:
• Avaliar f(x) em um valor específico (identificar qual regra usar)
• Esboçar o gráfico (desenhar cada pedaço no intervalo correto)
• Encontrar valores onde a função muda (pontos de fronteira)
• Determinar se a função é contínua nas fronteiras
Exemplo com Solução

Seja:

f(x) = { 2x + 1, se x < 2  |  x² − 1, se x ≥ 2 }

  • f(0) → usa 2x + 1 → 2(0) + 1 = 1
  • f(2) → usa x² − 1 → (2)² − 1 = 3
  • f(5) → usa x² − 1 → (5)² − 1 = 24
  • Continuidade em x = 2: lado esquerdo = 2(2) + 1 = 5, lado direito = 4 − 1 = 3. Como 5 ≠ 3, há uma descontinuidade em x = 2.
Aplicação Real — Tarifas

Uma empresa de eletricidade cobra:

C(kWh) = { 0.15 × kWh, se kWh ≤ 200  |  30 + 0.25 × (kWh − 200), se kWh > 200 }

Para 300 kWh: C(300) = 30 + 0.25 × 100 = R$55,00

Módulo 10

Assíntotas (Asymptotes)

Linhas que o gráfico se aproxima infinitamente mas nunca atinge

Conceito

Tipos de Assíntotas

Uma assíntota (asymptote) é uma linha reta que o gráfico de uma função se aproxima cada vez mais, sem nunca tocá-la (em geral). São cruciais para entender o comportamento de funções racionais e exponenciais.

Assíntota Horizontal
y = L, onde L = lim f(x) quando x → ±∞

O valor que f(x) se aproxima quando x se torna muito grande ou muito pequeno.

Exemplo: Para f(x) = 3 · (0.5)ˣ, a assíntota horizontal é y = 0, pois quando x → ∞, 0.5ˣ → 0.

No IB: Funções exponenciais f(x) = a · bˣ + c têm assíntota horizontal em y = c.

Assíntota Vertical
x = a, onde f(x) → ±∞ quando x → a

Ocorre quando o denominador de uma fração se torna zero.

Exemplo: Para f(x) = 1/(x − 2), a assíntota vertical é x = 2, pois o denominador é zero quando x = 2.

No IB: Para encontrar assíntotas verticais, resolva denominador = 0.

Assíntota Oblíqua
y = mx + c (reta inclinada)

Ocorre quando o grau do numerador é exatamente 1 a mais que o do denominador.

Exemplo: Para f(x) = (x² + 1)/x = x + 1/x, a assíntota oblíqua é y = x, pois 1/x → 0 quando x → ∞.

💡
Dica IB Exam: Na prova, assíntotas geralmente aparecem em:
• Funções exponenciais (assíntota horizontal)
• Funções racionais como 1/x (ambas: vertical e horizontal)
• Modelos reais com valores limite (ex: capacidade máxima de um rio)
Sempre indique assíntotas como linhas tracejadas no seu esboço!
Módulo 11

Análise de Funções (Function Analysis)

Comportamento, intervalos de crescimento/decrescimento e concavidade

Conceito

Comportamento Completo de uma Função

A análise completa de uma função envolve descrever todas as suas características: domínio, imagem, zeros, intervalos de crescimento, intervalos de decrescimento, concavidade e comportamento nos extremos.

Uma função é crescente em um intervalo quando, ao aumentar x, f(x) também aumenta. É decrescente quando, ao aumentar x, f(x) diminui.

No IB: Use a GDC para encontrar os turning points e depois descreva os intervalos. Exemplo: f(x) = x³ − 3x é crescente em (−∞, −1) ∪ (1, ∞) e decrescente em (−1, 1).

Côncava para cima (concave up, ∪): a curva "segura água", a taxa de variação está aumentando.

Côncava para baixo (concave down, ∩): a curva "derrama água", a taxa de variação está diminuindo.

O ponto onde a concavidade muda é o ponto de inflexão (point of inflection).

O "end behavior" descreve o que acontece com f(x) quando x → ±∞:

• Linear (grau 1): f(x) → ±∞ dependendo do sinal de m

• Quadrática (grau 2): ambos os lados vão na mesma direção (a > 0: ↑↑, a < 0: ↓↓)

• Cúbica (grau 3): lados opostos (a > 0: ↙↗, a < 0: ↗↙)

• Exponencial: um lado → assíntota, outro → ±∞

Exemplo — Análise Completa (IB Style)

Analise f(x) = −x³ + 3x² − 1

  • Domínio: x ∈ ℝ
  • y-intercept: f(0) = −1
  • Zeros: encontre com GDC ≈ −0.53, 0.65, 2.88
  • Turning points: mínimo local em (0, −1), máximo local em (2, 3)
  • Crescente: (0, 2)
  • Decrescente: (−∞, 0) ∪ (2, ∞)
  • Ponto de inflexão: (1, 1)
  • End behavior: x → −∞, f(x) → +∞ | x → +∞, f(x) → −∞
Checklist de Análise — IB Exam
① Domínio & Imagem   ② Interceptos (x e y)   ③ Assíntotas
④ Turning Points   ⑤ Crescente/Decrescente   ⑥ End Behavior

Use esta checklist em qualquer questão que peça para "analisar" ou "esboçar" uma função.

Módulo 12

Pontos Notáveis (Turning Points)

Identificando máximos, mínimos e pontos de inflexão em gráficos

Conceito

O que são Turning Points?

Turning points (pontos de retorno) são os pontos onde o gráfico muda de direção — isto é, onde a função passa de crescente para decrescente (máximo local) ou de decrescente para crescente (mínimo local).

Máximo Local (Local Maximum)

O ponto mais alto em uma vizinhança. A função cresce antes e decresce depois.

Mínimo Local (Local Minimum)

O ponto mais baixo em uma vizinhança. A função decresce antes e cresce depois.

Ponto de Inflexão (Point of Inflection)

Onde a concavidade muda. A curvatura inverte de "côncava para cima" para "côncava para baixo" ou vice-versa.

💡
Regra prática para o IB:
• Uma função de grau n pode ter no máximo n − 1 turning points
• Quadrática (grau 2) → máximo 1 turning point
• Cúbica (grau 3) → máximo 2 turning points
• Na calculadora GDC, use minimum e maximum para encontrá-los
Exemplo

Para f(x) = x³ − 6x² + 9x + 1

  • Máximo local em x = 1: f(1) = 5
  • Mínimo local em x = 3: f(3) = 1
  • Ponto de inflexão em x = 2: f(2) = 3
  • A função é crescente em (−∞, 1) ∪ (3, +∞)
  • A função é decrescente em (1, 3)
Módulo 13

Modelagem e Regressão (Modeling & Regression)

Usando funções para descrever fenômenos do mundo real a partir de dados

Conceito

O que é Modelagem Matemática?

Modelagem (mathematical modeling) é o processo de usar funções matemáticas para representar situações do mundo real. A partir de dados observados, escolhemos a função que melhor se ajusta aos dados.

Processo de Modelagem (Passo a Passo):

1
Coletar dados — Observar e registrar os valores do fenômeno
2
Plotar os dados — Criar um diagrama de dispersão (scatter plot)
3
Escolher o modelo — Observar o formato dos dados para escolher o tipo de função
4
Regressão — Usar calculadora/software para encontrar a equação que melhor se ajusta
5
Avaliar — Verificar o coeficiente de determinação (R²) e fazer previsões

Resumo do Módulo

Todos os conceitos fundamentais em um só lugar

Funções

  • Uma regra: entrada → saída única
  • Notação: f(x)
  • Domínio, Contradomínio, Imagem
  • Teste da reta vertical

Linear

  • f(x) = mx + c
  • m = inclinação, c = y-intercept
  • Paralelas: m₁ = m₂
  • Perpendiculares: m₁ · m₂ = −1

Quadrática

  • f(x) = ax² + bx + c
  • Vértice: x = −b/(2a)
  • Δ = b² − 4ac
  • Fórmula de Bhaskara

Cúbica

  • f(x) = ax³ + bx² + cx + d
  • Até 2 turning points
  • Sempre 1+ raiz real
  • Braços em direções opostas

Exponencial

  • f(x) = a · bˣ
  • b > 1 → crescimento
  • 0 < b < 1 → decaimento
  • Assíntota horizontal

Inversa

  • f⁻¹ "desfaz" f
  • Troque x ↔ y e isole
  • Reflexão sobre y = x
  • f(f⁻¹(x)) = x

Transformações

  • y = a·f(b(x−c))+d
  • c = shift horizontal, d = vertical
  • −f(x) reflete no eixo x
  • f(−x) reflete no eixo y

Compostas

  • (f∘g)(x) = f(g(x))
  • Aplique g primeiro, depois f
  • Ordem importa: f∘g ≠ g∘f
  • f(f⁻¹(x)) = x (identidade)

Por Partes

  • Regras diferentes por intervalo
  • Verifique qual regra usar
  • Teste continuidade nas fronteiras
  • Tarifas escalonadas, impostos

Assíntotas

  • Horizontal: y = L (x → ±∞)
  • Vertical: x = a (denominador = 0)
  • Exponencial: y = c em a·bˣ + c
  • Desenhe como linhas tracejadas

Análise

  • Crescente vs Decrescente
  • Concavidade: ∪ vs ∩
  • End behavior por grau
  • Checklist: D, I, Z, A, TP, EB

Modelagem

  • Scatter plot → escolher modelo
  • Regressão: linear, quad, exp
  • R² próximo de 1 = bom ajuste
  • Interpolação vs Extrapolação
